Element


Element är olika funktioner som finns inom html t.ex h1 taggar som är text eller div.


HTML


Html använder man när man vill skapa websidor. Man använder html för att strukturera en websida med rubriker text etc.


ID


ID-attributet för ett element är en identifierare som måste vara unik i hela dokumentet. Syftet är att identifiera elementet på ett unikt sätt när du länkar, skript eller styling.


Class


Class används för att sätta namn på olika Element. Man kan använda css för att styla denna specifika div class eller kombinera med java script.


Meta


Meta beskrivning är utdrag av text som beskriver en sidans innehåll. Meta beskrivningen visas inte på själva sidan, endast i sidans kod.


Responsive


Responsive web design är ett sätt att låta webbsajtens layout automatiskt anpassa sig efter betraktarens skärms bredd.


Http


Är ett protokoll man använder när man ska föra över data genom internet


Https


Https är ett likandant protokoll fast man krypterar informationen för att säker ställa så ingen annan kan ta den och läsa den.


Dns


Är ett inernetprotokoll som ger namn till din ip och hittar vilken adress du kommer ifrån


CDN


Med cdn avses geografiskt utspridda nätverk av proxy-servrar och deras datorhallar. Avsikten med dessa nätverk är att tillhandahålla nättjänster nära användare, för ökad driftsäkerhet och prestanda samt minskad latens.


Compressed data


Datakompression är när data omkodas på ett sätt som gör att färre informationsbärande enheter behöver användas.


Local bootstrap


Bootstrap Protocol är ett UDP-protokoll som används av en nätverksklient för att erhålla en IP adress från en BOOTP server på det lokala nätverket. BOOTP protokollet används ofta under bootningsprocessen hos datorer eller deras operativsystem.


JQuery


Är ett java script biblotek som sätter ihop rader i din kod för att få det mindre komplicerat


Flexbox


Flexbox används för att strukturera hemsidor, man använder den genom att skapa så kallad flexboxar där man manipulerar vart bilder text etc sitter.


Grid


Grid är likt flexbox och skapar rutnät eller en rutnättslayout som du kan använda för att manipulera din hemsida.


Code snippets


En snippet är ett block med text som infogas på lämpligt ställe men det innehåller även en del logik, så det betydligt bättre än att bara klippa och klistra.


WASM


WebAssembly definierar ett portabelt binärkodformat och ett motsvarande textformat för körbara program såväl som mjukvarugränssnitt för att underlätta interaktioner mellan sådana program.


Chronium


Chromium är en webbläsare skapad från öppen källkod på vilken Google Chrome, Microsoft Edge och Torch är baserade.


svg


SVG är ett bildformat vars innehåll är uppbyggt av vektorgrafik. Det innebär att du behöver specialiserade program för att kunna redigera en SVG-fil som bild.


png24


PNG24 formatet har stöd för 24 bitarsfärg. PNG 24 har i likhet med JPEG-formatet stöd för 24 bitarsfärg så att de subtila variationerna i ljusstyrka och nyanser som finns i fotografierna bevaras
